###git常用指令

####git status 
    
    判断当前文件或者文件夹的状态，是否有无进行修改操作   


####git pull

     从远程服务器拉取最新的代码，可以理解为本地代码更新到最新版本   


####git add  文件名/文件夹名

    提交修改的文件或者文件夹


####git commit -m  

    对修改的文件或者文件夹进行说明，相当于是添加了文档注释


####git push 

    将代码提交到远程服务器


####git log 
    
    查看代码的提交记录


####git reset --hard 版本号

    将版本回滚到某一历史版本(版本号就是对应历史版本的记录号，可以通过git log来获取所有的提交版本号)
